HTML/CSS Cheatsheet

HTML

HTML tags and elements form the building blocks of every website on the internet today.
HTML elements determine the general layout of a website's content.

Tag Name Description
<a> Anchor The Anchor element, with its href attribute, creates a hyperlink to web pages, files, email address, locations in the same page or anything else a URL can address.
<abbr> Abbrevation The Abbreviation element represents an abbreviation or acronym.
<ul> Unordered List The Unordered List element represents an unordered list of items, typically rendered as a bulleted list.
<ol> Ordered List The Ordered List element represents an ordered list of items, typically rendered as a numbered list.
<li> List Item The List Item is used to represent an item in a list.
<script> Script The Script element is used to embed executable code or data; this is typically used to embed or refer to JavaScript code.

Semantic HTML

Tag Name Description
<article> Article The Article element specifies independant, self-contained content.
<aside> Aside The Aside element defines some content aside from the content it is placed in (like a sidebar).
<details> Details The Details element defines additional details that the user can view or hide.
<figure> Figure The Figure tag specifies self-contained content, like illustrations diagrams, photos, code listings, etc.
<figcaption> Figcaption The Figcaption tag defines a caption for a Figure element.
<footer> Footer The Footer element defines a footer for a document or section.
<header> Header The Header element represents a container for introductory content or a set of navigational links.
<main> Main The Main element specifies the main content of a document.
<nav> Navigation The Nav element defines a set of navigational links.
<section> Section The Section element defines a section in a document.

CSS

The basic goal of the Cascading Stylesheet (CSS) language is to allow a browser engine to paint elements of the page with sepcific features, like colors, positioning, or decorations.

Attribute Name Syntax Description
display Display The CSS display property is specified using keyword values. The CSS display property sets whether an element is treated as a block or inline elemenmt and layout used for its children, such as flow layout, grid or flex.
align-content Align Content The CSS align-content property is specified using keyword values. The CSS align-content property sets the distrobution of space between and around content items along a flexbox's cross-axis or a grid's block axis.
align-items Align Items The CSS align-items property is specified using keyword values. The CSS align-items property sets the align-self value on all direct children as a group. In Flexbos, it controls the alignment of items on the cross axis.
justify-content Justify Content The CSS justify-content property is specified using keyword values. The CSS justify-content property defines how the browser distributes space between and around content items along the main-axis of a flex container, and the inline axis of a grid..
justify-items Justify Items The CSS justify-items property is specified using keyword values. The CSS justify-items property defines the default justify-self for all the items of the box, giving them all a default way of justifying each box along the appropriate axis.
margin Margin The CSS margin property is specified using numerical values. The CSS margin propertysets the margin area on all four sides of an element, creating extra space around an element. See Box Model
padding Padding The CSS padding property is specified using numerical values. The CSS padding propertysets the margin area on all four sides of an element, creating extra space within an element. See Box Model
